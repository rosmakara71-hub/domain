រកលក្ខខណ្ឌដើម្បីឱ្យអនុគមន៍ $f(x) = \frac{x}{\sin x + 1}$ មានន័យ។ (ចំពោះ $k \in \mathbb{Z}$) (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
លក្ខខណ្ឌ៖ $\sin x + 1 \neq 0 \Rightarrow \sin x \neq -1$។ តម្លៃមុំដែលធ្វើឱ្យ $\sin x = -1$ គឺ $x = -\frac{\pi}{2} + 2k\pi$ (ឬ $x = \frac{3\pi}{2} + 2k\pi$) ដូចនេះលក្ខខណ្ឌគឺ $x \neq -\frac{\pi}{2} + 2k\pi$។
រកលក្ខខណ្ឌដើម្បីឱ្យអនុគមន៍ $f(x) = \frac{1}{\sin^2 x - \cos^2 x}$ មានន័យ។ (ចំពោះ $k \in \mathbb{Z}$) (៥ ពិន្ទុ)

តាមរូបមន្តត្រីកោណមាត្រ គេមាន $\cos(2x) = \cos^2 x - \sin^2 x$។ ដូចនេះភាគបែងអាចសរសេរជា៖ $-(\cos^2 x - \sin^2 x) = -\cos(2x)$។ លក្ខខណ្ឌភាគបែងខុសពីសូន្យ៖ $-\cos(2x) \neq 0 \Rightarrow 2x \neq \frac{\pi}{2} + k\pi \Rightarrow x \neq \frac{\pi}{4} + \frac{k\pi}{2}$។
រកដែនកំណត់នៃអនុគមន៍ $f(x) = \frac{1}{1 - \tan x}$ ក្នុងចន្លោះ $[0, \pi]$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អនុគមន៍នេះមានលក្ខខណ្ឌពីររួមគ្នា៖
1. អនុគមន៍ $\tan x$ ខ្លួនឯងត្រូវមានន័យ នាំឱ្យ $x \neq \frac{\pi}{2}$។
2. ភាគបែងរួមត្រូវខុសពីសូន្យ៖ $1 - \tan x \neq 0 \Rightarrow \tan x \neq 1 \Rightarrow x \neq \frac{\pi}{4}$។
ដូចនេះក្នុងចន្លោះ $[0, \pi]$ ដែនកំណត់ត្រូវដកទាំងតម្លៃ $\frac{\pi}{4}$ និង $\frac{\pi}{2}$ ចេញ។
